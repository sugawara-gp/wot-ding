# Installation

```
npm install
```

# Run locally

```
npm start
```

# Environment Variables

- `PORT` : (optional) port number of this server. default is 3000
- `SLACK_TOKEN` : (require) Slack Outgoing Webhook's token
- `SLACK_WEBAPI_TOKEN` : (require) Slack Web API's token (https://api.slack.com/web)
